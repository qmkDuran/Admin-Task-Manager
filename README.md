<h1>Admin-Task-Manager 📝</h1><h2>Project Overview</h2><p><strong>Admin-Task-Manager</strong> is a tool designed to help administrative professionals keep track of their tasks. The application allows users to add tasks with text descriptions or images, helping them avoid forgetting important tasks during their busy workday.</p><h3>Key Features:</h3><ul><li><strong>🗂️ Task Management</strong>: Users can add tasks with a title, description, and optional image.</li><li><strong>📋 Task Organization</strong>: Tasks are organized into three sections: an add section, a task list section, and a done section.</li><li><strong>🔧 Task Actions</strong>:<ul><li><strong>➕ Add Button</strong>: Adds the task to the task list.</li><li><strong>✅ Done Button</strong>: Moves the task to the done section.</li><li><strong>🗑️ Delete Button</strong>: Removes the task entirely.</li><li><strong>🔄 Undone Button</strong>: Moves a task from the done section back to the task list section.</li><li><strong>✏️ Edit Button</strong>: Allows users to update the text and image of a task in the task list, with a save button to update the task.</li></ul></li><li><strong>ℹ️ About Me Section</strong>: Provides information about the project creator.</li></ul><h2>Tech Stack 🛠️</h2><h3>Frontend</h3><ul><li><strong>⚛️ React</strong>: Used to build the user interface.</li><li><strong>📡 Axios</strong>: Handles HTTP requests to the backend.</li><li><strong>🧪 Testing Libraries</strong>:<ul><li><strong>@testing-library/react</strong>: Used for testing React components.</li><li><strong>@testing-library/jest-dom</strong>: Provides custom Jest matchers for testing DOM nodes.</li><li><strong>@testing-library/user-event</strong>: Simulates user interactions in tests.</li></ul></li><li><strong>🌐 Cross-Environment</strong>: Managed through <code>cross-env</code> to ensure consistency of environment variables across different platforms.</li><li><strong>⚙️ React-Scripts</strong>: Used for configuration and scripts that create a React app.</li></ul><h3>Backend</h3><ul><li><strong>🚀 NestJS</strong>: A progressive Node.js framework for building efficient and scalable server-side applications.</li><li><strong>🗄️ TypeORM</strong>: An ORM (Object-Relational Mapping) library used with TypeScript and JavaScript (ES7, ES6, ES5).</li><li><strong>🐘 PostgreSQL</strong>: The primary database used to store application data.</li><li><strong>📦 Multer</strong>: Middleware for handling <code>multipart/form-data</code>, used for uploading files.</li><li><strong>🔗 RxJS</strong>: A library for reactive programming using Observables, which makes it easier to compose asynchronous and callback-based code.</li><li><strong>🧪 Jest</strong>: A testing framework used for unit tests and end-to-end testing.</li><li><strong>🚨 Supertest</strong>: A library for testing HTTP requests.</li></ul><h3>Development Tools 🛠️</h3><ul><li><strong>🔷 TypeScript</strong>: A strongly typed programming language that builds on JavaScript, giving you better tooling at any scale.</li><li><strong>💻 ESLint &amp; Prettier</strong>: Tools used for maintaining code quality and formatting consistency.</li><li><strong>🛠️ Nest CLI</strong>: A command-line interface for managing the NestJS project lifecycle.</li></ul><h3>Database This will move the task to the "Task List" section.</li></ul><h3>Task List Section</h3><ul><li><strong>📋 View Tasks</strong>: Tasks that you’ve added will appear here. Each task displays the title, description, priority, and an image if one was uploaded.</li><li><strong>✏️ Edit Task</strong>: Click the "Edit" button to modify the task. You can change the title, description, priority, and image. After saving, the new text and priority will update immediately, but the new image will only appear after you refresh the page.</li><li><strong>✅ Done Task</strong>: Click the "Done" button to move the task to the "Done" section.</li><li><strong>🗑️ Delete Task</strong>: Permanently remove the task by clicking the "Delete" button.</li></ul><h3>Done Section</h3><ul><li><strong>📦 View Completed Tasks</strong>: Tasks that have been marked as done will appear in this section.</li><li><strong>🔄 Undo Task</strong>: If you want to move a task back to the "Task List" section, click the "Undo" button.</li><li><strong>🗑️ Delete Task</strong>: You can also permanently delete a task from this section. Note that tasks in the "Done" section cannot be edited.</li></ul><h3>About Me Section</h3><ul><li><strong>ℹ️ Access</strong>: The "About Me" button is located at the bottom left corner of the page. It’s a blue button that, when clicked, opens a modal window.</li><li><strong>🛠️ Information</strong>: The modal provides information about the project creator, including experience and a link to their LinkedIn profile.</li><li><strong>❌ Close</strong>: To close the modal, click the "X" button in the top right corner.</li></ul><h3>Task Management and Layout</h3><ul><li><strong>📜 Scrolling Feature</strong>: If you create multiple tasks, the sections (Enter Task, Task List, and Done) do not expand.
